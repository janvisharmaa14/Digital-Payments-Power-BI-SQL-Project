Overview

This project analyzes a digital payments transaction dataset using Power BI and SQL. It focuses on deriving insights from transaction data and presenting them through an interactive dashboard.

Tools Used

* Microsoft Power BI
* SQL

Key Features

* Transaction analysis by recharge type
* Monthly trend analysis
* Payment success vs failure insights
* Interactive dashboard with filters and navigation

 Key Insights

* Utility-based transactions contribute significantly to overall volume
* Majority of transactions are successful
* Certain categories generate higher transaction values
* Monthly trends highlight variations in user activity
